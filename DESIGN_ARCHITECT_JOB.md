## üìã Responsibilities

### 1. Visual System Design
- Create comprehensive style guide for all game assets
- Define color palettes, typography, and visual hierarchy
- Establish lighting and material standards
- Design particle effects and animation standards

### 2. Asset Production Planning
- Catalog all required assets (characters, cards, UI, effects)
- Prioritize MVP vs. polish assets
- Create technical specifications for each asset type
- Design asset pipeline from concept to integration

### 3. UI/UX Design
- Design Hearthstone-level game interface
- Create wireframes and high-fidelity mockups
- Design responsive layouts for mobile/desktop
- Create design system (components, tokens, patterns)

### 4. Art Direction
- Review and approve AI-generated assets
- Provide feedback on consistency and quality
- Create paint-overs or refinements where needed
- Ensure visual cohesion across all elements

### 5. Implementation Support
- Export assets in correct formats
- Create CSS/component specifications
- Work with developers on integration
- Iterate based on in-game testing
