## üöÄ Generation Commands

Generate all prompts using:

```bash
# Titan Portraits
curl -X POST http://localhost:8080/chat -H "Content-Type: application/json" -d '{"prompt": "Generate image: [PROMPT_TEXT]"}'

# Or use the Python client
python3 << 'EOF'
from orchestrator.client import chat

prompts = [
    "Epic fantasy character portrait, muscular orc warlord...",
    # ... more prompts
]

for prompt in prompts:
    response = chat(f"Generate image: {prompt}")
    print(response)
EOF
```
